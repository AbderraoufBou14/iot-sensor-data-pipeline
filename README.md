# IoT Sensor Data Pipeline

Projet de simulation et traitement de données capteurs avec ingestion dans PostgreSQL et visualisation dans Grafana.

## Fonctionnalités
- Génération de données IoT simulées (température, humidité, pression)
- Ingestion automatique dans PostgreSQL
- Visualisation via Grafana

## Technologies utilisées
- Python
- PostgreSQL
- Docker & Docker Compose
- Grafana
- SQL

## Démarrage
1. Clonez le dépôt
2. Lancez : `docker-compose up --build`
3. Accédez à Grafana sur `http://localhost:3000` (admin/admin)
